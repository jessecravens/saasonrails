saasonrails

Build your app, not billing code.

Build Fast

Speed up Development and Deployment. Avoid writing subscription management, recurring billing, and credit card management code for your next Ruby On Rails project.

Stripe Payments Integration

SaaSonRails gets your new software-as-a-service site off to a running start by providing you a well-tested and proven recurring billing system with Stripe payments integration.

And more ...

You also get a framework for building membership-based applications, including basecamp-like subdmomains, account signups/upgrades/downgrades, tiered pricing levels with customizable limits, multi-tenant data security in a single database.
